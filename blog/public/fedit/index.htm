Using FEdit

First, download FEdit using this link. (87KB)
Note: You need to have Java installed on your computer for FEdit to work. Get Java Here!
Now unzip the files using a program such as WinZip.

Go to the folder where you unzipped FEdit, and double-click on "fgui.bat":

This screen will display:

Enter the hostname of the FTP server for your website.
Enter the username and password for your FTP.

If you have Microsoft FrontPage or another website authoring program, enter the path to it int the html Editor and .htm Editor textboxes. Otherwise, enter the path to Notepad, which is probably either C:\Windows\Notepad.exe or C:\WINNT\Notepad.exe.

Click "Save Changes".

You have now setup FEdit. You don't need to do this again.

Eventually, a list if files will come up, like this:

To edit a file, just double-click it. The file will be downloaded and an editing program will start.

To change directories, just double-click the directory name.
